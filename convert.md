Before you hit "Clone" Make sure you know where your minecraft launcher keeps its modpacks, here are mine in a folder called "instances"
	
<img width="732" alt="Screen Shot 2021-12-30 at 7 45 16 AM" src="https://user-images.githubusercontent.com/34868944/147754296-4ef4e707-2cf9-4400-9d34-1738a27917fe.png">


Now click "Choose" under the "Local path" option and point github desktop to the folder where your modpacks are stored (works for any launcher).
	
<img width="480" alt="Screen Shot 2021-12-30 at 7 45 58 AM" src="https://user-images.githubusercontent.com/34868944/147754286-7689db97-b233-4edb-8091-78177c107aee.png">
	
Now click clone and Github Desktop will create a new folder inside the folder you pointed it to, it will then place all the project files in that folder which in this case create a modpack. Now restart your launcher and you will see the new modpack added.
	
Whenever you make changes to any files in that modpack folder they will appear in a list in github desktop.
	
<img width="249" alt="Screen Shot 2021-12-30 at 7 50 47 AM" src="https://user-images.githubusercontent.com/34868944/147754339-55fe14b2-82fe-4281-a02a-27dd29b9bbc3.png">

	 
from there you must give a short summary of your changes and if you wish you can provide a longer description of them, then hit the "Commit to main" button.
	 
Github will now give you the option to "push" your changes (this is uploading them so other devs can download the changes)
	
  <img width="962" alt="Screen Shot 2021-12-30 at 7 53 10 AM" src="https://user-images.githubusercontent.com/34868944/147754369-03e3d285-7107-4b0e-a403-bd662267ade4.png">
	  
sometines you may see a "Pull origin" in that case someone else has made changes you should download. We reccomend usign the "Fetch origin" (Check for changes) and then checking for the "pull" button every time you start working on a project but Github is even capable of pushing your changes and pulling others at the same time.
